预定义的媒体查询

Name Media Query
default 默认屏幕（所有尺寸有效）
small 所有尺寸屏幕有效
small-only 在 0-640px 分辨率有效
medium 仅在 641px 以上分辨率有效
landscape 横屏模式
portrait 竖屏模式
retina 适合 iphone 4 以上的视网膜屏
retina3x 适合 iphone 6 plus 及其它 3x 高清屏

自定义媒体查询名称

  $(document).mobile('responsive', {
    named_queries: {
      my_custom_query: 'only screen and (max-width: 200px)'
    }
  });

事件

  $(document).on('replace', 'img', function(e, new_path, original_path) {
    console.log(e.currentTarget, new_path, original_path);
  });

当页面中添加了新的响应式内容

  $(document).mobile('responsive', 'reflow');
