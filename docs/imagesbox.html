照片专辑

如果你希望在页面上只显示一张图片，在查看大图里切换多图，或者想在照片上加些文字介绍，可以这么做：

其它可选项

  $(document).mobile({
    imagesbox: {
      // specify the classes or IDs will close clearing when clicked.
      close_selectors : '.imagesbox-close',

      // specify the classes or IDs that will open clearing when clicked, defaults to the entire li element
      open_selectors : '',

      // specify the classes or IDs that will be skipped in carousel
      skip_selector : '',

      touch_label : '← Swipe to Advance →'
    }
  });

可用的事件列表

  • open.imagesbox before a thumbnail is expanded
  • opened.imagesbox after the large image has opened
  • close.imagesbox before the large image is closed
  • closed.imagesbox after the large image has closed
  • change.imagesbox when the large image changes
  • resized.imagesbox after the large image is resized, before it becomes visible

当页面中添加了新的图片查看器

  $(document).mobile('imagesbox', 'reflow');
