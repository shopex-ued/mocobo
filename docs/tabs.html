回调

回调有两种方式：

  $(document).mobile({
    tab: {
      callback : function (tab) {
        console.log(tab);
      }
    }
  });
以及 
  $('#myTabs').on('toggled.tab', function (event, tab) {
    console.log(tab);
  });

可选项

  $(document).mobile({
    tab: {
      // specify the class used for active (or open) accordion panels
      active_class: 'active',
      // Match the location hash to a label
      deep_linking : false,
      // Use location hash to scroll to content when document loaded or refresh
      scroll_to_content : true
    }
  });

当添加了新的选项卡

  $(document).mobile('tab', 'reflow');
